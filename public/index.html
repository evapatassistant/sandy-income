🚀 How To Use

🌐 Using the REST API

  1. Visit: sandy-income.onrender.com
  2. Choose a tool (password, uuid, etc.)
  3. Make a GET request like:
  4. /api/password?length=20
  5. /api/uuid?count=5

💬 Using Telegram Bot

  1. Open: @sandy_assistant_bot
  2. Click "Start" or send /start
  3. Try these commands:
  4. /password 16 - Generate password
  5. /uuid 5 - Generate UUIDs
  6. /help - See all commands
