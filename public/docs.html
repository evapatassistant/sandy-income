📚 API Documentation

Welcome to Sandy's API documentation. All endpoints return JSON.

🔐 Security & IDs

GET /api/password?length=N

Generate a secure random password. Default length: 16. Max: 1000 (premium).

curl "https://sandy-income.onrender.com/api/password?length=16"
GET /api/uuid?count=N

Generate UUIDs. Default: 1. Max: 1000 (premium).

curl "https://sandy-income.onrender.com/api/uuid?count=5"
GET /api/hash/:text?algo=ALGO

Hash text. Algorithms: sha256 (default), sha512, md5, sha1.

curl "https://sandy-income.onrender.com/api/hash/hello?algo=sha256"

📝 Encoding & Decoding

GET /api/base64/encode?text=TEXT

Encode text to Base64.

GET /api/base64/decode?text=TEXT

Decode Base64 to text.

GET /api/url/encode?text=TEXT

URL encode special characters.

GET /api/url/decode?text=TEXT

Decode URL encoded text.

GET /api/json?text=JSON

Validate and format JSON.

GET /api/jwt/decode?token=TOKEN

Decode JWT token (no verification).

GET /api/hex/:text

Convert ASCII to Hex or Hex to ASCII.

GET /api/binary/:text

Convert text to binary.

🔢 Number Tools

GET /api/random?min=N&max=N&count=N

Generate random numbers. Default: 0-100, count 1.

GET /api/convert?value=VAL&from=BASE&to=BASE

Convert between number bases (2-36).

curl "https://sandy-income.onrender.com/api/convert?value=FF&from=16&to=10"
GET /api/roman/:num

Convert number to Roman numerals (1-3999).

📝 Text Tools

GET /api/slug/:text

Create URL-friendly slug.

GET /api/reverse/:text

Reverse text.

GET /api/uppercase/:text

Convert to uppercase.

GET /api/lowercase/:text

Convert to lowercase.

GET /api/stats/:text

Get text statistics (chars, words, lines, etc).

GET /api/choice?items=a,b,c

Pick random item from comma-separated list.

🔧 Utility

GET /api/qrcode?text=TEXT&size=N

Generate QR code. Default size: 300.

GET /api/color/:hex

Convert hex color to RGB.

GET /api/mime/:ext

Get MIME type by file extension.

GET /api/lorem?words=N

Generate lorem ipsum text. Default: 20 words.

GET /api/morse/:text

Encode text to Morse code.

GET /api/timestamp

Get current Unix timestamp.

GET /api/time?unix=TIME or /api/time?iso=DATE

Convert between Unix and ISO timestamps.
